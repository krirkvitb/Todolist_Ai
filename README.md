# Todolist_Ai

โปรแกรมบรรทัดคำสั่ง (CLI) สำหรับจัดการ To-Do List แบบง่าย ๆ เขียนด้วย Python

**คุณสมบัติหลัก**
- เพิ่มงานใหม่ (title, description, due_date)
- ดูงานทั้งหมด
- แก้ไขงาน (title, description, completed)
- ลบงาน (ยืนยันก่อนลบ)
- บันทึก/โหลดข้อมูลจากไฟล์ `tasks.json`

**Prerequisites**
- Python 3.8 ขึ้นไป ติดตั้งไว้ในระบบ

**ติดตั้ง / เตรียมรัน**
1. เปิด PowerShell แล้วเข้าโฟลเดอร์โปรเจ็กต์:

```powershell
cd ...\AIcode\Todolist_Ai
```

2. (ถ้าต้องการ) สร้าง virtual environment:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

3. (ไม่จำเป็น) ติดตั้ง dependencies ถ้ามี — โครงการนี้ไม่ใช้แพ็กเกจภายนอก

**รันโปรแกรม**

```powershell
python ...\AIcode\Todolist_Ai\main.py
```

เมื่อรันจะเห็นเมนูหลักให้เลือกตัวเลข (1-5)

**ตัวอย่างการใช้งาน (ตัวอย่างอินเทอร์แอคทีฟ)**

- เพิ่มงานใหม่: เลือก `1` → ใส่ `ชื่อเรื่อง`, `รายละเอียด`, `วันครบกำหนด` → โปรแกรมจะแสดงข้อความยืนยัน
- ดูงานทั้งหมด: เลือก `2` → โปรแกรมจะแสดงรายการงานที่เก็บในหน่วยความจำ/ไฟล์
- แก้ไขงาน: เลือก `3` → เลือกลำดับงาน (index) → แก้ `title`/`description`/`completed` (y/n)
- ลบงาน: เลือก `4` → เลือกลำดับงาน → ยืนยันลบด้วย `y` เพื่อยืนยัน
- ออกจากโปรแกรม: เลือก `5` หรือกด `Ctrl+C` → โปรแกรมจะบันทึก `tasks.json` ก่อนออก

**ข้อมูลการจัดเก็บ**
- ไฟล์ข้อมูลอยู่ที่: `tasks.json` (สร้างอัตโนมัติเมื่อบันทึกครั้งแรก)
- รูปแบบ: JSON array ของ object แต่ละงานมี key: `id`, `title`, `description`, `due_date`, `completed`

**ข้อเสนอแนะเพิ่มเติม**
- หากต้องการให้ข้อมูลเก็บในที่อื่น (เช่น `%APPDATA%` หรือฐานข้อมูล SQLite) สามารถปรับเพิ่มได้
- ถ้าต้องการฟีเจอร์เพิ่มเติม เช่น ค้นหา/กรองงานตามวันที่หรือสถานะ บอกได้เลย

---
ไฟล์หลักของโปรเจ็กต์: `main.py` — แก้ไขง่ายและอ่านได้สำหรับขยายฟีเจอร์เพิ่มเติม
